resumen: "Este proyecto presenta el diseño y validación de un sistema de telemonitorización de signos vitales en tiempo real para pacientes de la tercera edad. Utiliza un microcontrolador ESP32 y sensores de bajo costo para adquirir datos de ECG, SpO2, frecuencia cardíaca y temperatura. Implementa procesamiento en el borde (edge computing) para optimizar la transmisión de datos a una plataforma IoT, permitiendo una supervisión médica remota constante y eficiente."
